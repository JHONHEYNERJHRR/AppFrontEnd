# AppFrontEnd
En este proyecto voy a poner a pruebas mis conocimientos de los diferentes cursos que voy a estar tomando  en FrontEnd

Funcionalidad voy a crear una App Movil para llevar el control de un pago de tarifas de un taxi donde el conductor y el propietario o la empresa llevan el control de todos los movimientos de dinero que se realizan diarios por cuenta de las tarifas del vehiculos o los diferentes gastos que este tiene mes a mes por concepto de reparaciones o remplazo de piezas desgastadas como lo son las llantas o la bateria.
